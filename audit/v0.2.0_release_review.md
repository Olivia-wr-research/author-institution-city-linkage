## Test Results

- Original baseline tests: 20 passed.
- Final tests: 38 passed.
- Temporary virtual environment: install, tests, and `all-demo` passed.
- Dependency review: `pytest>=8` moved from runtime dependencies to the `test` optional dependency.
- Runtime install verified with `python -m pip install -e .`, `python -m scopus_toolkit --help`, and `python -m scopus_toolkit all-demo`.
- Test install verified with `python -m pip install -e ".[test]"` and `python -m pytest -q`.
- Final pre-push review found CRLF/LF byte differences between generated CSV files and v0.2.0 expected outputs.
- CSV writing now explicitly fixes `lineterminator="\n"` for deterministic LF output.
- v0.2.0 `all-demo` outputs and expected outputs are byte-identical without changing data content, fields, row order, column order, or business logic.

## Known Limitations

- Live Scopus adapter behavior is deferred.
- Synthetic source comparison is field-level only and not ground truth.
- Cognitive proximity is descriptive and not causal.
