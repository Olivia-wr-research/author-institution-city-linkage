# Scopus API And Data Boundaries

The public demos use `MockTransport`; they do not call Scopus or OpenAlex.

No credential file, API cache, raw response directory, or live output database is included. `SCOPUS_API_KEY` is documented only as an environment-variable name for future optional work.

Users must independently confirm that their use of Scopus, OpenAlex, or any other source complies with applicable terms, licenses, institutional requirements, and privacy rules.
